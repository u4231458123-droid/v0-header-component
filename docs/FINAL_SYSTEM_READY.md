## 11. Wichtige Hinweise

### 11.1 Linter-Warnungen
Die verbleibenden Linter-Warnungen ("Context access might be invalid") sind **normal** und **nicht kritisch**. Sie entstehen, weil der Linter nicht sicher ist, ob die Secrets in GitHub gesetzt sind. Dies ist in GitHub Actions normal - Secrets werden zur Laufzeit geladen.

### 11.2 Secrets setzen
Bitte setzen Sie die folgenden Secrets in GitHub Repository Settings:
- `HUGGINGFACE_API_KEY`
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`
